# 聖杯日記 — 繁體中文版(The Grail Diary of Henry Jones)

《聖戰奇兵》當年隨遊戲附贈的實體手冊《亨利·瓊斯的聖杯日記》,是亨利老瓊斯三十餘年(1898–1937)追尋聖杯的私人日記與蒐證簿,也是遊戲三道試煉謎題(上帝的氣息 / 話語 / 道路、自獅頭一躍)的線索本。字幕與配音都中文化了,這本手冊是最後一塊拼圖。

本目錄收錄它的**繁體中文版**:把英文全部換成中文,做成一本彷彿原生的中文聖杯日記,版面與美術完全比照原版。

## 忠實重建的版面元素

- **綠色封面 / 封底**:取原版掃描色值 `#1C6B2B`;封底保留原版權聲明。
- **雙欄考據正文**:襯線中文(Noto Serif CJK TC),頁眉沿用原版 `INDIANA JONES AND THE LAST CRUSADE` / `LUCASARTS CLASSIC ADVENTURES` 字距小型大寫。
- **打字機信件**(框內,等寬字):al-Jawf、埃莉諾拉夫人、施陶比希三封來信。
- **羅馬電報**:重建 `Ufficio Telegrafico di ROMA` 表格(義大利文表頭照留),電文中文化(科里羅利報告鐵達尼號將攜熱那亞保羅日記訪美)。
- **報紙剪報**:〈新發現福音書真偽受質疑〉(亞歷山卓路透社電)。
- **手繪插圖**:第 12 頁的鐘擺齒輪機關 + 棋盤地磚,直接重用原版線稿。

## 翻譯原則

- **英文敘述、信件、報導全部中文化**,語氣維持亨利老瓊斯 1898–1937 的學究日記口吻。
- **保留亨利「引用」的外語原始文獻片段**:拉丁文(*lapsit excellis*、`EQUESTRI SEPULCRUM IN URBE REGINA MARIS DALMATIAE`、`CHRISTI CALIX`)、亞蘭文、義大利電報表格欄位——即使中文版,亨利抄錄的原始引文仍照留,兼顧考據與遊戲謎題對照。
- **人名/地名/書名**沿用台灣慣譯與本專案術語(印第、亨利·瓊斯、馬可斯·布洛迪、唐納文、施奈德、加拉哈德爵士、《帕齊瓦爾》…)。
- 完整收錄全 13 頁、每一則日記條目,無刪節。

## 來源與作法

底本為隨遊戲附贈之 Grail Diary 手冊(LucasArts/Lucasfilm 版權);本中文版為同人中文化,僅供個人保存與致敬,不作商業用途。流程:抽原文文字層 → 對照掃描影像校正 → 繁中翻譯 → HTML/CSS 重建版面 → chrome-headless 輸出 A5 PDF。
